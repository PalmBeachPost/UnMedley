Welcome to UnMedley - the cure to all your Medley woes. Or at least a few of them.

This is a framework built on top of Medley to help you quickly create custom presentations for your stories. UnMedley, at its heart, is a CSS/JS framework. But you don't have to know CSS or javascript to churn out story layouts.

This is a work in progress. You can add templates or use existing ones. You can contribute to the code or pretend this is just magic.

You can help me write help documents or just yell at me when it doesn't work. Use it as you please.

This is one example of a story template. Let us call this a-fullscreen-section template. Download this (link TBD) file to get started.

Photos and Asides

Now that you kind of got a hang of how text works here, let's deal with images.

UnMedley, in this version, has two main options for your 'bonus' content. Two is better than one right? You can show the image as a full width display. Or as an aside. An aside pulls the content to the left or right as you specify. It does not have to be just images. More on that later.

First let us look at the main photo display option. Main content option lets you display the image fullwidth like the one shown below. You need to tell UnMedley that it is a photo. You do that with a class name and guess what? The class name for that is photo.

We fancy. We got cutlines here. The cutline can be longer and you can give photo credits also here.

As you can see, photo class also gives you an option to put a cutline with it. And the class for that is, you guessed it, cutline

This is an example of an image with the classes set to aside, pullout and right

But what if I want the photo to the side? UnMedley's got you covered. Let us call that an aside.

To make an aside, create a photo div as we did above. Then just wrap it in a parent div with the class name aside and say left or right to denote which side to display it on.

Here is what an aside looks like when pulled to the right. See how the photo slighty projects out further than the text? You have an option to do that. Add the class pullout to the aside to make it stand out just a tad bit.

Next is an example of pulling left and this time, let us not use pullout clas sto see the difference.

Please note, on a phone or any smaller screen, asides will behave as main content because there really isn't enough space to display an aside.
